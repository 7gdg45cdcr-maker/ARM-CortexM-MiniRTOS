# ARM Cortex-M Mini RTOS + Device Driver Framework

## Project Overview

This project implements a lightweight Real-Time Operating System (RTOS) for ARM Cortex-M microcontrollers using Embedded C. The RTOS is developed from scratch to understand the internal working of task scheduling, context switching, interrupt handling, synchronization mechanisms, and device driver development.

The project demonstrates practical embedded systems concepts commonly used in firmware development for automotive, IoT, industrial automation, and real-time applications.
